Présentation

Intégrateur web

Ayant un diplôme dans le domaine informatique, je m'efforce de réintégrer le monde professionnel en suivant une première formation dans le parcours OpenClassrooms en vue de devenir Full stack developer pour saisir de futures opportunités en mode freelance ou salarié. De nature curieuse, soucieuse du détail et rigoureuse, j'ai pu acquérir des compétences clés utilisées dans l'implémentation des projets de sites web qui respectent les attentes des clients. Cependant, je ne suis pas parfaite. Je suis très loin de la perfection, mais je suis résiliente, déterminée et je trouve que l'apprentissage continu est un processus fascinant.

Mes Travaux

Booki

Booki est un projet réalisé dans le cadre de ma formation d'intégrateur Web chez OpenClassrooms. J'ai travaillé à partir d'une maquette fournie pour intégrer la page d'accueil d'une agence de voyage fictive en utilisant les technologies HTML et CSS.

Ohmyfood

ohmyfood est un projet que j'ai réalisé dans le cadre de ma formation d'intégrateur Web chez OpenClassrooms. À partir d'une maquette fournie, j'ai travaillé sur l'intégration de la version mobile d'un site de foodtech avec des animations CSS.

Kasa

Kasa est une application que j'ai développée dans le cadre de ma formation. J'ai réalisé le développement d'une plateforme de location d'appartements en utilisant: HTML, CSS, JS, ainsi que React et React Router pour créer des interfaces dynamiques. J'ai également utilisé une API pour récupérer les données.

Argent Bank

Argent Bank est un projet que j'ai réalisé dans le cadre de ma formation. J'ai développé une application web avec authentification des utilisateurs en utilisant:HTML, CSS, JS, React, React Router, Redux, une API pour récupérer les données et MongoDB pour stocker les informations des utilisateurs.

Nina Carducci

Nina Carducci est un site d'une photographe que j'ai fait son optimisation de la performance,en faisant un meilleur référencement SEO pour plus de visibilité pour ces potentiels clients.

Agence 724events

Agence 724events est une agence évènementielle réalisé avec React que j'ai fait son débuggage, dans le cadre de ma formation,en utilisant Chrome DevTools. J'ai également rédiger un cahier de recette pour tester ce site.
